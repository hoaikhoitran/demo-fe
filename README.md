# demo-fe

Sustainable living landing page built with **React**, **TypeScript**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

## Scripts

- **Install dependencies**: `npm install`
- **Start dev server**: `npm run dev`
- **Build for production**: `npm run build`
- **Preview build**: `npm run preview`
- **Lint**: `npm run lint`

## Tech stack

- **Frontend**: React 19 + TypeScript
- **Build tool**: Vite 7
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion

## Project structure

- `src/App.tsx`: Root application component
- `src/components/layout`: Shared layout components (`Navbar`, `Footer`)
- `src/components/sections`: Landing sections (`Hero`, `Testimonials`, `Contact`, etc.)
- `src/hooks`: Reusable hooks like `useFadeInOnScroll`
- `src/context`: Global providers like `ThemeContext`
